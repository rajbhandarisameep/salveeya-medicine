# Execution Scripts

This directory contains **deterministic Python scripts** that handle the actual work:

- API calls
- Data processing
- File operations
- Database interactions

## Guidelines

1. Scripts should be **reliable, testable, and well-commented**
2. Use environment variables from `.env` for secrets and configuration
3. Each script should do **one thing well**
4. Always check here before writing a new script — reuse existing tools when possible
